# Assets Directory

This folder contains media assets for the MathVizAI project.

## Structure

```
assets/
├── screenshots/          # App screenshots for README
│   ├── pdf-viewer.png   # PDF viewer interface
│   ├── ocr-mode.png     # OCR selection mode
│   ├── video-gen.png    # Video generation process
│   └── animation.png    # Generated animation example
├── demo.gif             # Main demo GIF for README
└── README.md            # This file
```

## Adding Screenshots

1. Take screenshots of the app in action
2. Recommended dimensions: 800x600 or 1200x800 pixels
3. Save as PNG for quality or GIF for animations
4. Name files according to the structure above

## Creating a Demo GIF

You can use tools like:
- [ScreenToGif](https://www.screentogif.com/) (Windows)
- [Gifski](https://gif.ski/) (macOS)
- [Peek](https://github.com/phw/peek) (Linux)

Recommended settings:
- Resolution: 800x600 or smaller
- Frame rate: 10-15 fps
- Duration: 15-30 seconds
- Show key features: OCR selection, video generation, playback
